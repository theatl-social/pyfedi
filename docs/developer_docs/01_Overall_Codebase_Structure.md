# Navigating the PieFed Codebase

PieFed is an open source project that welcomes contributions from the wider open source community. In an effort to help ease contributions to the project, this developer documentation aims to help familiarize prospective contributors with how the codebase for the project is organized so that features and fixes can be implemented easier and quicker.

For support or questions about the project, some code, or contributions, feel free to reach out to the project organizers in any of these ways:

- Join the zulip server at [chat.piefed.social](https://chat.piefed.social/).
- Join the [matrix chat for PieFed development](https://matrix.to/#/#piefed-developers:matrix.org).
- Create an issue in the [PieFed Codeberg repository](https://codeberg.org/rimu/pyfedi/issues).
